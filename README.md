### Punks XL

Upsize "classic" 24px punks to XL using "matching" cyberpunks attributes in 32px.

Example - Case Study - Female 1/2/3, Male 1/2/3 and The First Six Punks Side-By-Side (in 4x)

![](cyberpunks-classic/i/punks-xl@4x.png)
